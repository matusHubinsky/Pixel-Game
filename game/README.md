# Pixel=Game

Simple 2D pixel game written in C using SDL library 

# Controls
 
<pre>
	==============================================
	Movement:
	==============================================
	W       move up 
	A       move left  
	S       move down 
	D       move right

	==============================================
	Combat:
	==============================================
	Up      attack 
	Down    attack 
	Left    attack 
	Right   attack 

	==============================================
	Debug Optios:
	==============================================
	1       draw all blocks         (blue)
	2       draw all edges          (red)
	3       draw all visible lines  (green)

	==============================================
	Other:
	==============================================
	Esp     close game
	Space   respawn 
</pre>
