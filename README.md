# Pixel-Game

Simple 2D pixel game written in C using SDL library 

# Controls


<p> W       move up <br> </p> 
<p> A       move left <br> </p> 
S       move down <br>
D       move right <br>

Up      attack <br>
Down    attack <br>
Left    attack <br>
Right   attack <br>

Esp     close game <br>
Space   respawn <br>
